У меня возникают проблемы при просмотре фильма

Попробуйте следующее:

  • Зайдите в Настройки
  • Отключите параметр Блокировать рекламу
  • Попробуйте посмотреть фильм

При любом поисковом запросе нету результатов

Для поиска фильмов и сериалов, приложение использует API КиноПоиска.

Если у вас случается подобная ситуация, это чаще всего означает что сервера, которые отвечают за работу API, временно не работают.

Попробуйте подождать около часа и повторить поиск снова.
