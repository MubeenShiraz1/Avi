У мене виникають проблеми під час перегляду фільмів

Спробуйте наступне:

  • Зайдіть у Налаштування
  • Відключить параметр Блокувати рекламу
  • Спробуйте знову подивитись фільм

При будь-якому пошуковому запиті немає результатів

Для пошуку фільмів і серіалів, додаток використовує API сервiсу "Кинопоиск".

Якщо у вас трапляється подібна ситуація, це найчастіше означає що сервера, які відповідають за роботу API, тимчасово не працюють.

Спробуйте почекати близько години і повторити пошук знову.
